# CodeAlpha_StockPortfolioTracker

## Description

This project is a simple stock portfolio tracker developed in Python. Users enter stock names and quantities, and the program calculates the total investment value using predefined stock prices stored in a dictionary.

## Features

* Stock selection
* Quantity input
* Investment calculation
* Total portfolio value calculation
* Easy terminal-based interface

## Technologies Used

* Python
* VS Code

## Concepts Used

* Dictionary
* Loops
* User Input
* Arithmetic Operations
* Variables

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:

python stock_tracker.py

## Sample Input

AAPL
4

TSLA
2

done

## Sample Output

AAPL: $180 x 4 = $720
TSLA: $250 x 2 = $500

Total Investment Value = $1220
